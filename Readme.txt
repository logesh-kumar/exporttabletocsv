You can export any html tables to CSV with our shortcode.

== Description ==

Place this short code [ETCSV] YOUR BUTTON TEXT [/ETCSV] just below any HTML table. This shortcode will Enable option to download that particular table as a CSV file.

== Installation ==


1.  Upload the entire export-tables-to-csv folder to the /wp-content/plugins/ directory
2.  Activate the plugin through the 'Plugins' menu in WordPress
